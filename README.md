# 🔥 Logic Master Pro

> Interactive Visual Lab for Digital Logic Gates & IC Simulation

---

## 🧠 Overview

**Logic Master Pro** is a modern, web-based interactive lab designed to help students understand **digital logic gates** in a visual and practical way.

It combines **theory + visualization + simulation** in one place, making it easier to learn and experiment with logic circuits.

---

## 🎯 What This Project Includes

### 🟢 Logic Gates Visualization

* AND Gate
* OR Gate
* NOT Gate
* NAND Gate
* NOR Gate
* XOR Gate
* XNOR Gate

Each gate includes:

* 📌 Logic Symbol
* 🔌 IC Pinout (74 Series)
* 📊 Truth Table

---

### 🔌 IC Integration (74 Series)

The project demonstrates real-world ICs such as:

* 7408 (AND)
* 7432 (OR)
* 7404 (NOT)
* 7400 (NAND)
* 7402 (NOR)
* 7486 (XOR)
* 74266 (XNOR)

This helps bridge the gap between **theoretical logic** and **practical electronics**.

---

### ⚡ Truth Table Generator (Simulator)

An interactive simulator that allows users to:

* Enter logical expressions
* Generate truth tables dynamically
* Test logic circuits instantly

Example:

```text
(A & B) | C
```
